
| ----------- | ---------------------------------------- | ------------------------------------ |
| Method Name | Description                              | Short Example                        |
| ============| ======================================== | =====================================|
| append()    | Adds an element at the end of the list   | `a=[1,2]; a.append(3)` → `[1,2,3]`   |
| ----------  | ---------------------------------------- | ------------------------------------ |
| clear()     | Removes all the elements from the list   | `a=[1,2]; a.clear()` → `[]`          |
| ----------- | ---------------------------------------- | ------------------------------------ |
| copy()      | Returns a copy of the list               | `a=[1,2]; b=a.copy()`                |
| ----------- | ---------------------------------------- | ------------------------------------ |
| count()     | Returns number of occurrences of a value | `a=[1,2,2]; a.count(2)` → `2`        |
| ----------- | ---------------------------------------- | ------------------------------------ |
| extend()    | Adds elements of another iterable        | `a=[1]; a.extend([2,3])` → `[1,2,3]` |
| ----------- | ---------------------------------------- | ------------------------------------ |
| index()     | Returns index of first occurrence        | `a=[1,2,3]; a.index(2)` → `1`        |
| ----------- | ---------------------------------------- | ------------------------------------ |
| insert()    | Adds element at a specific position      | `a=[1,3]; a.insert(1,2)` → `[1,2,3]` |
| ----------- | ---------------------------------------- | ------------------------------------ |
| pop()       | Removes & returns element at index       | `a=[1,2]; a.pop()` → `2`             |
| ----------- | ---------------------------------------- | ------------------------------------ |
| remove()    | Removes first matching value             | `a=[1,2,2]; a.remove(2)` → `[1,2]`   |
| ----------- | ---------------------------------------- | ------------------------------------ |
| reverse()   | Reverses list order                      | `a=[1,2]; a.reverse()` → `[2,1]`     |
| ----------- | ---------------------------------------- | ------------------------------------ |
| sort()      | Sorts list in ascending order            | `a=[3,1,2]; a.sort()` → `[1,2,3]`    |
| ----------- | ---------------------------------------- | ------------------------------------ |
